# Project Journey: How We Built This Website

Here is a step-by-step overview of the development process and technologies used to build this platform:

1. **Frontend Creation** 
   - We kickstarted the project by building the initial frontend using AI, which helped us quickly set up a modern and responsive user interface.

2. **Initial Database Setup**
   - We integrated a local **MySQL** database as our initial data store to manage users, orders, and other application data.

3. **Cloud Database Migration**
   - For better scalability, reliability, and management, we later migrated our MySQL database to the cloud using **Aiven**.

4. **AI Chatbot Integration**
   - To enhance customer support and user interaction, we integrated a smart conversational chatbot powered by the **Gemini API**.

5. **Demand Forecasting Model**
   - We implemented advanced analytics by training a Machine Learning model using Random Forest Regressor. This model helps in accurately forecasting demand to optimize inventory and operations.
